# StudentSuccessPredictor

Веб-застосунок на **ASP.NET Core MVC** для ведення журналу студентів та **прогнозування успішності** за допомогою **ML.NET**.

## Опис предметної області
У застосунку ведеться журнал студентів (вік/стать/професії батьків/попередній бал/пропуски/нездачі/репетитори/бажання вищої освіти).
На основі цих даних модель машинного навчання прогнозує очікувану успішність студента.

## Основний функціонал
- Журнал студентів: додавання/редагування/видалення записів (без БД, збереження у JSON-файл).
- Прогноз успішності:
    - прогноз для одного студента;
    - прогноз для всіх студентів;
    - перенавчання моделі на основі датасету UCI + власних даних (всі дані вигадані).

## Технології
- ASP.NET Core MVC
- ML.NET (`Microsoft.ML`, `Microsoft.ML.FastTree`)
- Збереження даних у файлах ([App_Data/journal.json](cci:7://file:///C:/Users/aleks/StudentSuccessPredictor/App_Data/journal.json:0:0-0:0))
- Датасет: UCI Student Performance (додатково доповнений власними даними)

## Запуск
1) Клонувати репозиторій:
   git clone <repo_url>
   cd StudentSuccessPredictor
2) Очистити збірку (опційно):
   dotnet clean
3) Запустити застосунок:
   dotnet run
